# 🎰 幸运大转盘

一个精美的网页版幸运大转盘抽奖游戏！

## ✨ 特性

- 🎨 精美的渐变色转盘设计
- 💡 闪烁灯泡环效果
- 🎊 中奖撒花动画
- ⚖️ 可自定义奖品和概率
- 📊 抽奖历史记录
- 📱 完美适配手机端
- 🚀 纯前端，无需后端

## 🎁 默认奖品

| 奖品 | 概率 |
|------|------|
| 🏆 特等奖 | 2% |
| 🎁 一等奖 | 5% |
| 🧧 二等奖 | 10% |
| 🎫 三等奖 | 15% |
| 🍭 参与奖 A | 18% |
| 🍪 参与奖 B | 18% |
| 🥤 谢谢参与 | 16% |
| 🌈 再来一次 | 16% |

## 🔧 自定义奖品

编辑 `index.html` 中的 `PRIZES` 数组即可修改奖品和概率。

## 📦 部署

本项目为纯静态页面，可直接部署到 GitHub Pages、Vercel、Netlify 等平台。
